# 更新履歴

このファイルは、アプリケーションの主要な変更点を記録するものです。

---

## 2025-09-04 - v1.15.0

### 追加・修正
- **サカイ販路のガス情報反映**: サカイ販路で「すまいのでんき」とガスセットを選択した際、ガスの開栓日と時間帯が営業コメントに正しく反映されるように修正しました。ガスセットでない場合は、これまで通り「ガス：なし」と表示されます。
- **全販路共通コメントタグの体裁修正**: 全ての営業コメントにおいて、ヘッダー内の識別タグ（`250811`）の前に半角スペースが挿入されるように修正し、体裁を整えました。
- **キューエネスでんきの法人契約機能追加**:
  - 電気タブの「キューエネスでんき」選択時、備考欄の下に「法人契約」チェックボックスを追加しました。
  - チェックを入れると、「対応者（漢字）」「対応者（フリガナ）」の必須入力欄が表示されます。
  - 入力された対応者情報は、営業コメントの末尾に自動で追記されるようにしました。

## 2025-09-03 - v1.14.0

### 変更・修正
- **名乗り自動設定の分離**: 電気・ガスタブでレコードIDが `ID:` で始まる場合に「名乗り」が「すまえる」に自動設定された後、インターネットタブに切り替えると名乗りが引き継がれず空欄になるように修正しました。
- **ヘッダー表示の変更**: ヘッダーの「最終更新日時」を「バージョン表示」に変更しました。
- **開発ルールの更新**: 上記2点の変更を`memory/rules.md`に反映しました。

## 2025-09-02 - v1.13.0

### 追加・変更
- **東急ガスの現住所反映**: `ガス`タブで「東急ガス」を選択した場合、営業コメントに郵便番号（例: `〒123-4567`）を含む現住所が正しく表示されるように修正しました。
- **電気3商材の特典備考**: `電気`タブで「すまいのでんき」「プラチナでんき」「キューエネスでんき」のいずれかを選択し、かつレコードIDが `ID:` または `CC`+数字で始まる場合、備考欄に自動で「5,000円CB」が入力される機能を追加しました。ユーザーが編集した場合はその内容が保持されます。
- **インターネットタブの名乗り修正**: レコードIDが `ID:` で始まる場合に「名乗り」が「すまえる」に自動設定される機能が、電気・ガスタブでのみ有効になるよう修正しました。インターネットタブで直接入力しても自動設定されなくなります。
- **すまいのガスの契確時間フォーム追加**: `ガス`タブで「すまいのガス」を選択し、「契確あり」の場合に、営業コメントに反映される「契確時間」の入力欄をフォームに追加しました。
- **最終更新日時の更新**: ヘッダーの最終更新日時を `2025/09/02 10:00` に更新しました。

## 2025-09-01 - v1.12.0
